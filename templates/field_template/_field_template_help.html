Field templates are groups/sets of fields that can be defined and set to an inventory. These fields will be used when adding new items.

For example, you could define the following sets of fields for an inventory of computers:

  • Model
  • Serial number
  • Manufacturer
  • Hard drive capacity
  • ...

These fields would then be available to fill in when you add new items to that inventory.

You can add a new inventory by clicking the icon at the top.

You can specify the sort order of the template fields using the icon.
