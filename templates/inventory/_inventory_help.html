Lists are collections of objects forming a common theme, such as your collection of retro computers your partner hates, or your collection of Mongolian tooth flossing sticks.

You can add a new list by clicking the icon at the top. Along with the list name and description, you can specifiy whether the list will be private (default) or allow item level visibility. Item level visibility allows items within the list to be publically viewable. By default, items are created private.

The icon indicates that the list is private (to you). The indicates an list with item level visibility.

Items (or more precisely things) can be moved between lists from within the lists themselves.

In addition to user defined fields and templates, items within lists will, by default display a few default fields such as type, location, tags and images. If you dont want to show these on the item pages for this list you can turn them off on the Add or Edit list form.
